## /resources/charts/<RegExp>/source/src

NMEA2000 src value or any similar value for encapsulating the original source of the data

* Type: `string`
* Path: `/resources/charts/(^[A-Za-z0-9_-]{8,}$)/source/src`
* Node: `src`

### Example:
```
36
```
