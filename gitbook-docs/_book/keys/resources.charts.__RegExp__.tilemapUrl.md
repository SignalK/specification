## /resources/charts/<RegExp>/tilemapUrl

A url to the tilemap of the chart for use in TMS chartplotting apps

* Type: `string`
* Path: `/resources/charts/(^[A-Za-z0-9_-]{8,}$)/tilemapUrl`
* Node: `tilemapUrl`

### Example:
