## /resources/charts/<RegExp>/name

Chart common name

* Type: `string`
* Path: `/resources/charts/(^[A-Za-z0-9_-]{8,}$)/name`
* Node: `name`

### Example:
```
NZ615 Marlborough Sounds
```

### Source:
```
{
  "type": "string",
  "description": "Chart common name",
  "example": "NZ615 Marlborough Sounds"
}
```
