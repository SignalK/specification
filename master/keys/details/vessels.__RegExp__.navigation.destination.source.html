/vessels//navigation/destination/source

Source of data, a record of where the data was received from. An object containing at least the properties defined in 'properties', but can contain anything beyond that.

  • Type: object
  • Path: /vessels/(^urn:mrn:(imo|signalk):(mmsi:[2-7][0-9]{8,8}|uuid:[0-9A-Fa-f]{8}-[0-9A-Fa-f]{4}-4[0-9A-Fa-f]{3}-[89ABab][0-9A-Fa-f]{3}-[0-9A-Fa-f]{12}))|^(http(s?):.*|mailto:.*|tel:(\+?)[0-9]{4,})$/navigation/destination/source
  • Node: source

Source:
